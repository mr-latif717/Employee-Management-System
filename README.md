## Project Objective

The Employee Management System is a responsive web application developed using HTML, CSS, and JavaScript. It helps HR departments manage employee information through a dashboard, employee registration form, employee list, and company information page.

---

## Features

- Dashboard
- Employee Registration Form
- JavaScript Form Validation
- Employee List
- Search Employee by Name
- Filter by Department
- About Company Page
- Responsive Design
- Dark Mode
